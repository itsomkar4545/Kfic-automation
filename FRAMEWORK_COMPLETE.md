## 🛠️ **Installation & Setup**

### **Quick Setup**
```bash
# 1. Basic setup
pip install -r requirements.txt

# 2. Advanced setup (with all features)
setup_advanced.bat

# 3. Run tests
run_advanced.bat
```

### **Manual Advanced Setup**
```bash
# Install advanced dependencies
pip install -r requirements_advanced.txt

# Install Allure (requires Node.js)
npm install -g allure-commandline

# Verify installation
robot --version
allure --version
```

## 🎮 **How to Use**

### **1. GUI Mode (Easiest)**
```bash
python gui_executor.py
# OR
run_gui.bat
```

### **2. Command Line**
```bash
# Quick test
robot tests/reliable_login_test.robot

# Enhanced test with Allure
robot --listener allure_robotframework tests/enhanced_login_test.robot

# Generate Allure report
allure serve results/allure-results
```

### **3. Advanced Runner**
```bash
run_advanced.bat
# Choose from 8 different execution modes
```
